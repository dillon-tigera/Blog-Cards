The Problem
Why Are Organizations Moving Away from VMware?
Broadcom's acquisition of VMware has driven significant licensing changes. Enterprises are re-evaluating their virtualization strategy and looking to Kubernetes as a modern alternative.
HOVER FOR MORE →
Key Details
  • Licensing uncertainty — Broadcom's acquisition led to pricing and bundling changes
  • Kubernetes as the target — consolidate VMs onto K8s to unify operations
  • Networking is the blocker — pod networking doesn't support existing IPs or VLANs
  • Re-addressing is painful — forces updates to LBs, DNS, and firewalls
  • VIP workarounds add complexity — turns migration into a network redesign
  • The gap — K8s can run VMs, but the network doesn't move with the workload
The Platform
What Is KubeVirt?
KubeVirt extends Kubernetes to run traditional virtual machines alongside containers, enabling teams to manage both workload types with the same tools and workflows.
HOVER FOR MORE →
Key Details
  • VMs as K8s objects — defined as CRDs, managed with kubectl and GitOps
  • Unified operations — one toolchain for both containers and VMs
  • Live migration — move VMs between nodes without downtime
  • Lift-and-shift ready — import existing VM disk images directly
  • Networking challenge — default pod networking assigns new IPs, breaks VLANs
  • Why Calico matters — L2 Bridge Networks close the networking gap
The Solution
What Is Calico L2 Bridge Mode?
Calico L2 Bridge Networks extend existing Layer 2 infrastructure directly into Kubernetes, preserving original IP addresses and VLANs during VM migration.
HOVER FOR MORE →
Key Details
  • L2 network extension — bridge on cluster nodes extends external networks into K8s
  • IP preservation — VMs keep original addresses, no re-addressing needed
  • Multi-VLAN support — multiple VLANs via trunk-backed bridges
  • eBPF observability — deep traffic visibility without external tooling
  • K8s-native policy — label-based network policy for VM interfaces
  • Phased migration — maintain L2 adjacency during transition periods
